rescale

collapse all

Rescale data to a specified range.

Syntax

Y = rescale(X)
Y = rescale(X, A, B)

Description

Y = rescale(X) rescales the elements of X to the default range [0, 1].

Y = rescale(X, A, B) rescales to the range [A, B].

Linearly rescales the elements of x so that they span the target range $[a, b]$. The minimum value of x maps to a and the maximum maps to b. Values between are linearly interpolated.

Examples

collapse all

Rescale to default range [0, 1]

x = [1 2 3 4 5];
y = rescale(x)

Rescale to a custom range

x = [1 2 3 4 5];
y = rescale(x, -1, 1)

Rescale columns of a matrix independently

M = rand(5, 3) * 100;
Y = zeros(size(M));
for k = 1:size(M, 2)
    Y(:,k) = rescale(M(:,k));
end

Input Arguments

collapse all

x — Input data array

Input data array. Input data, specified as a vector, matrix, or N-D array. All elements participate in determining the min and max used for scaling. If all elements of x are equal, the output contains NaN values due to division by zero; consider adding a guard if this case may arise.

double

a — Lower bound of target range

Lower bound of target range.

double

Default: 0

b — Upper bound of target range

Upper bound of target range. Upper bound of the target range. If b < a, the output is reversed (maximum of x maps to a).

double

Default: 1

Output Arguments

collapse all

y — Rescaled data.

Rescaled data. Rescaled data, returned as an array the same size as x with values in the range $[a, b]$.

Tips

  • For data with outliers, consider clipping before rescaling: x = max(min(x, upper), lower) before calling rescale.
  • To rescale each column of a matrix independently, loop over columns or use normalize(x, 'range') from Statistics and Machine Learning Toolbox.

Algorithms

The transformation applied is:

$$y = a + \frac{x - \min(x)}{\max(x) - \min(x)} \cdot (b - a)$$

This is a standard affine rescaling (min-max normalization).
